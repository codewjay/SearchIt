# SearchIt

A Desktop File Search Assistant developed using C++ and Qt.

## Features

- Exact Search
- Prefix Search
- Smart Search
- Extension Search
- Case Insensitive Searching
- Duplicate File Detection
- Dark Mode
- Export Results
- Recursive Directory Scanning

## Technologies Used

- C++
- Qt 6.11.1
- STL
- Trie Data Structure
- Object Oriented Programming
- File System Programming

## Installation

1. Download SearchIt_v1.0.zip
2. Extract the ZIP file.
3. Open SearchAssistant.exe.

## Search Modes

- Smart Search
- Exact Search
- Prefix Search
- Extension Search
